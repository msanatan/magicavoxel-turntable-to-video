# magicavoxel-turntable-to-video

Take MagicaVoxel's turntable images in a folder and make a video out of them.

MagicaVoxel's turntable feature generates a series of images in a folder. Sometimes you might want to convert it to a video for sharing on social media and the like.

## Dependencies

You need to install ffmpeg on your local machine for this package to work

## How to use

```
turntabletovideo <path to folder with images>
```
